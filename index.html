📌 Nasıl Çalışır?

AutoTube şu adımlarla içerik üretir:

  • 1. Konu belirlenir ve başlık oluşturulur
  • 2. GPT ile senaryo yazılır
  • 3. Metin otomatik seslendirilir
  • 4. Görsellerle birlikte video haline getirilir
  • 5. YouTube’a otomatik yüklenir

🛠️ Altyapı Teknolojisi

AutoTube sisteminde kullanılacak teknolojiler:

  • Frontend: HTML + CSS + JS (Vercel ile barındırılıyor)
  • Backend: Python (FastAPI ile) veya Node.js (Express)
  • AI: ChatGPT API ile senaryo oluşturma
  • Seslendirme: ElevenLabs API (veya Google Text-to-Speech)
  • Video Oluşturma: ffmpeg ile video birleştirme
  • Otomasyon: YouTube API ile otomatik yükleme
